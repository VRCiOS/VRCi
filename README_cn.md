# VRCi <img src="./icon/icon.png" width="64" height="64"> </img>

[🇺🇸 English](README.md) • [🇯🇵 日本語](README_ja.md) • [🇪🇸 Español](README_es.md) • [🇨🇳 中文](README_cn.md) • [🇰🇷 한국어](README_kr.md)

**VRCi** 是一款新一代好友与社区管理工具，其灵感源自 iOS 平台上的 [**VRCX**](https://github.com/vrcx-team/VRCX)，专为 VRChat 打造，并以 SwiftUI 精心构建。这款**非官方第三方应用程序**致力于在尊重 VRChat 生态系统的前提下，提升您的 VRChat 体验。

## 主要功能

### 🚀 **原生 iOS 体验**
- **无需浏览器**：通过原生 iOS App 即可直接访问 VRChat 的所有功能，告别 VRChat 官网。
- **好友位置即时追踪**：无需切换应用，实时了解好友动态。
- **闪电般迅捷的性能**：优化的原生 SwiftUI 界面，带来如丝般顺滑的操作体验。

### 👥 **更强大的好友与社群管理**
- **好友状态实时更新**：通过实时 WebSocket 更新，即时掌握好友的在线状态。
- **全面用户搜索**：借助强大的搜索与筛选功能，轻松找到任何 VRChat 用户。
- **收藏与群组管理**：轻松标记您喜爱的伙伴、世界和虚拟形象，并将他们整理到可自定义的群组中。

### 🌍 **世界与内容探索**
- **高级世界搜索**：通过精细的筛选、排序和推荐功能，探索全新世界。
- **内容浏览**：无需离开应用，即可探索虚拟形象、世界和群组。
- **心愿单管理**：保存并整理您希望稍后体验的内容。

### 📱 **无缝连接体验**
- **增强型二维码分享**：通过包含您的虚拟形象和信息的精美定制二维码，轻松分享您的 VRChat 个人资料。
- **二维码扫描器**：扫描好友的 VRCi 二维码即可快速添加，无需手动输入。
- **多账户支持**：通过安全的凭证管理，轻松存储并切换多达 3 个 VRChat 账户。

### ⚙️ **专业个人资料工具**
- **全面个人资料管理**：通过直观易用的界面，编辑您的个人资料、管理简介链接、语言和状态。
- **状态历史记录**：轻松查看并复用您之前的状态消息。
- **隐私控制**：随心管理您的可见性和互动偏好。

### 🎨 **现代化的用户界面与体验**
- **iOS 原生设计**：专为 iOS 精心打造的流畅直观界面，全面支持深色模式。
- **无障碍优先**：内置旁白和多项辅助功能，关怀每一位用户。
- **个性化体验**：根据您的偏好和使用习惯，自定义专属应用体验。

*更多精彩功能，敬请期待！*
